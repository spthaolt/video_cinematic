About Video Cinematic

video_cinematic is an oxwall plugin. Video Cinematic Mode allow your oxwall video player has ability to dimm the light around and center the video. If you love this plugin please buy me a beer (via paypal to thaolt@songphi.com). Don't hesitate to send me an email on how you want to improve this plugin or ideas of new features, etc.

Developing

git clone https://github.com/ycicom/video_cinematic.git

Making plugin package from source

cd /path/to/video_cinematic
  ./make-pkg.sh
